INTRODUCTION                                                       *Ambienter*

This plugin allow you to quickly switch betwen two predefined themes
(background style and colorscheme), manually or automatically.

:Ambienter                                                         *Ambienter*
    
    Switch ambient between 'dark' and 'light' theme.
    This command accept the bang (!) form (see |Ambienter!|)

:Ambienter!                                                        *Ambienter!*
    
    Like |Ambienter| but force the 'dark' theme.

:AmbienterSet {ambient}                                         *AmbienterSet*

    Set the theme with the {ambient} predefined values.
    {ambient} must be user defined.

:AmbienterSensor                                             *AmbienterSensor*

    Define automatically the current theme, based on ambient light detection
